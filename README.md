📘 水利双题库-云加载版 · 使用说明
第1版 · 双题库云加载版 — 简洁高效的水利安全知识练习与考试系统

一、快速入门
系统提供登录试用注册三种方式进入：

已有账号：输入邮箱和密码，点击登录即可进入。
免注册试用：点击免注册试用 (5题)，可直接体验5道题目，无需注册。
注册新账号：切换到“注册”标签，填写邮箱、手机号（将用作云错题库码）和密码，提交后按提示发送邮件给管理员审核。
💡 提示：登录后系统会自动回到上次离开时的题目，并优先加载您上次使用的题库，提供连贯的学习体验。
二、题库切换
系统内置第1题库和第2题库两套独立题库，您可以根据学习需求随时切换：

登录页或主界面顶部均可找到切换题库按钮。
切换后，练习和考试将使用对应题库的题目，错题库也会自动切换到对应的错题记录。
首次使用某个题库时，系统会从云端加载题目并缓存到本地，后续打开无需等待。
三、练习模式
登录后默认进入练习模式，这是最常用的刷题方式：

选择范围：点击全部题库练习所有题目，或点击错题库只练习曾做错的题目。
选择题型：点击单选多选判断切换题型。
作答与反馈：点击选项选中，点击确认提交答案，系统会即时显示对错和正确答案。
错题自动收集：在“全部题库”中答错的题目会自动存入错题库，方便后续集中攻克。
答对移除：在“错题库”中答对后，该题会自动从错题库中移除。
🎯 推荐用法：先用“全部题库”全面练习，系统自动收集错题；再用“错题库”集中消灭薄弱点。答对即移除，让错题本越做越薄！
四、考试模式
点击顶部考试模式标签进入模拟考试：

系统从题库中随机抽取单选100道 + 多选20道 + 判断20道组成试卷（试用模式仅5道）。
计分规则：单选/判断每题0.5分；多选每题2分，漏选每个正确选项得0.5分（最高1.5分），错选不得分。
点击确认锁定答案，点击交卷查看总分和通过情况。
交卷后可点击错题回看查看本次考试的详细错题记录。
考试中的错题同样会自动存入错题库，供练习模式复习。
⚠️ 注意：若题库数据尚未完全加载到本地，考试功能会提示缺失的批次数量，请稍等片刻再尝试。
五、错题库与云备份
错题库数据优先保存在您的浏览器本地，并配有云端备份功能：

错题库码：由管理员分配，登录后自动填入（输入框为禁用状态），用于区分不同题库的错题存储。
刷新错题库：点击刷新错题库从云端同步最新错题数据。
备份到云端：点击错题备份到云端将本地错题手动上传至云端。
自动备份：退出账号或关闭页面时，系统会自动暂停并备份错题，请耐心等待备份完成。
📌 温馨提示
1 建议使用Chrome、Edge、Firefox等现代浏览器，以获得最佳体验。
2 试用模式有5题限制，注册登录后可解锁全部功能。
3 切换题库或退出时请确保错题备份完成，避免数据丢失。
4 如遇题库加载缓慢，请检查网络连接，系统已在后台持续缓存数据。
